# 🐾 Neural Microscope: Biological Morphological Classification
## 📌 Project Overview
This repository contains a professional-grade tutorial and implementation of a Convolutional Neural Network (CNN) for the classification of biological subjects. Using the **Cats vs. Dogs** dataset, this project explores how artificial neural computation mirrors the hierarchical processing of the mammalian visual cortex.
## 🎯 Learning Outcomes
This project satisfies the following academic criteria:
* **Neural Computation:** Modeling receptive fields and spatial hierarchies.
* **Advanced ML Methods:** Implementation of **Data Augmentation** and **Dropout**.
* **Professional Analysis:** Utilizing Confusion Matrices and class-wise diagnostics.
* **Ethical AI:** Addressing **Automation Bias** and the societal impact of AI.
## 📁 Repository Contents
* `Tutorial_Report.pdf`: A 2,000-word deep-dive into theory, ethics, and critical research evaluation.
* `CNN_Implementation.ipynb`: The complete Jupyter Notebook implementation using TensorFlow/Keras.
* `README.md`: Documentation and setup instructions.
* `LICENSE`: MIT License for open-source re-use.

pip install tensorflow matplotlib numpy scikit-learn
